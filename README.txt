DKPro WSD
=========

DKPro WSD is a modular, extensible Java framework for word sense
disambiguation.  It provides UIMA components which encapsulate corpus
readers, linguistic annotators, lexical semantic resources,
disambiguation algorithms, and evaluation and reporting tools.

This project contains DKPro WSD modules which are licensed under
the terms of the Apache License.  There is another project,
DKPro WSD GPL, which contains further modules licensed under the
terms of the GNU General Public License.

Both projects can be obtained from their websites on Google Code:

  https://code.google.com/p/dkpro-wsd/
  https://code.google.com/p/dkpro-wsd-gpl/
